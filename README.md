# TopicModeling_LDA

Latent Dirichlet Allocation (LDA) applied to 2,000 PubMed abstracts using collapsed Gibbs sampling to discover 20 latent biomedical topics.


See the full implementation and results in [LDA.ipynb](LDA.ipynb).
